地图示例。

## 何时使用

### 导入

```bash
$ import { Map } from '@sakitam-gis/react-map';
```

## API


|   参数    |   类型   |   默认  |   说明     |
|-----------|----------|------------|-------------------|
| className  | String   |  ''      | 容器类名 |
| center  | Array   |  null      | 地图中心点 |
| zoom  | Number   |  null      | 地图层级 |

> [查看详细使用](http://maptalks.org/maptalks.js/api/0.x/Map.html)
